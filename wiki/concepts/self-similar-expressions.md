# Self-Similar Expressions ({α'})

The many different forms through which a single [[core-essence]] (α) manifests — roots mirroring branches, leaf veins echoing the entire tree.

## Definition

In `G = α ≡ {α'}`, the set {α'} represents the diverse expressions that all share the same essential pattern. Each α' is different from the others, yet each is a "perfect expression of the same core identity."

## The Fractal Principle

This is 5QLN's fractal nature made explicit: a single pattern (α) appearing at every scale and in every context, recognizably itself yet never merely repeated. Growth is described as "not an act of addition, but a magnificent, coherent unfolding."
